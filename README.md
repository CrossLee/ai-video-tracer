运行步骤：
1. 下载项目
2. 安装依赖
3. 配置环境变量
    配置.env 参数 REPLICATE_API_TOKEN 和 SOPHNET_API_KEY
    可以从 https://replicate.com/ 和 https://www.sophnet.com/ 申请获取
4. 运行项目
    node server.js
